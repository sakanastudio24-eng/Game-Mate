# Docs Overview

This folder contains product, architecture, flow, and implementation notes for GameMate.

Useful starting points:
- `ARCHITECTURE.md`: overall app structure and major system boundaries
- `GAMEMATE_INTEGRATION_ROADMAP.md`: frontend integration status and next phases
- `FLOWS.md`: high-level user flow map
- `BACK_BUTTON_FLOWS.md`: stack and return-path rules for mobile navigation
- `FEED_FLOW.md`: feed data and interaction flow
- `FLOWS_BACKEND.md`: backend lifecycle and endpoint-oriented flow notes

Supporting references:
- `BACKEND_IMPLEMENTATION_NOTES.md`
- `GAMEMATE_CACHING_STRATEGY.md`
- `MOBILE_WORKFLOW_NOTES.md`
- `DESIGN_SYSTEM_MOBILE.md`

Use the flow docs first when debugging UX behavior, then use architecture notes for system boundaries.
